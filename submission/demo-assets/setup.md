# PromptGate Demo Setup

This file is a draft for the demo assets package.

## Prerequisites

- Local PromptGate proxy / app environment
- Local LFM runtime
- W&B run link for evaluation evidence

## Demo Flow

1. Start PromptGate.
2. Open the target GenAI tool workflow.
3. Send a sensitive prompt.
4. Show PromptGate blocking or returning an alternative response before cloud submission.
5. Show W&B `model_comparison` table.
